# OOTD — Style Inspo App

A Flutter fashion app with Firebase backend.

## Requirements

- Flutter SDK `^3.10.8`
- Dart SDK `^3.10.8`
- Android Studio / Xcode (for device/emulator)
- Firebase project with Auth, Firestore, Storage, and Cloud Functions enabled

## Flutter Dependencies

Defined in `pubspec.yaml`. Run `flutter pub get` to install.

| Package | Version | Purpose |
|---|---|---|
| firebase_core | ^4.6.0 | Firebase initialization |
| firebase_auth | ^6.3.0 | User authentication |
| cloud_firestore | ^6.2.0 | Database |
| firebase_storage | ^13.2.0 | Image storage |
| cloud_functions | ^6.1.0 | Gemini AI (StyleLab) |
| image_picker | ^1.1.1 | Camera / gallery access |
| uuid | ^4.5.2 | Unique IDs for uploads |
| cupertino_icons | ^1.0.8 | iOS-style icons |

## Firebase Setup

1. Create a Firebase project at [console.firebase.google.com](https://console.firebase.google.com)
2. Enable **Authentication** (Email/Password)
3. Enable **Cloud Firestore** (start in test mode, then apply rules below)
4. Enable **Firebase Storage**
5. Enable **Cloud Functions**
6. Download config files and place them:
   - `android/app/google-services.json`
   - `ios/Runner/GoogleService-Info.plist`
   - `lib/firebase_options.dart` (generated by `flutterfire configure`)

> These files contain API keys — they are in `.gitignore` and must NOT be committed.

## Firestore Security Rules

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    match /users/{userId} {
      allow read: if true;
      allow write: if request.auth != null && request.auth.uid == userId;

      match /likedPosts/{postId} {
        allow read, write: if request.auth != null && request.auth.uid == userId;
      }
      match /savedPosts/{postId} {
        allow read, write: if request.auth != null && request.auth.uid == userId;
      }
      match /following/{targetId} {
        allow read, write: if request.auth != null && request.auth.uid == userId;
      }
      match /followers/{followerId} {
        allow read: if request.auth != null;
        allow write: if request.auth != null;
      }
      match /plans/{planId} {
        allow read, write: if request.auth != null && request.auth.uid == userId;
      }
    }

    match /posts/{postId} {
      allow read: if request.auth != null;
      allow create: if request.auth != null;
      allow update, delete: if request.auth != null && request.auth.uid == resource.data.uid;

      match /likes/{userId} {
        allow read, write: if request.auth != null;
      }
      match /comments/{commentId} {
        allow read: if request.auth != null;
        allow create: if request.auth != null;
        allow delete: if request.auth != null && request.auth.uid == resource.data.uid;
      }
    }
  }
}
```

## Firebase Storage Rules

```
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      allow read, write: if request.auth != null;
    }
  }
}
```

## Running the App

```bash
# Install dependencies
flutter pub get

# Run on connected device or emulator
flutter run

# Build APK
flutter build apk --release
```

## Cloud Functions (StyleLab AI)

```bash
cd functions
npm install
firebase deploy --only functions
```
